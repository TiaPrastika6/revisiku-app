# RevisiKu

RevisiKu adalah aplikasi sederhana berbasis Python Streamlit untuk mencatat tugas, revisi, catatan bimbingan, praktikum, dan deadline mahasiswa.

## Fitur

- Dashboard ringkasan catatan
- Tambah catatan akademik
- Daftar catatan
- Filter berdasarkan kategori, status, dan prioritas
- Ubah status catatan
- Edit catatan
- Hapus catatan
- Penyimpanan data lokal menggunakan JSON

## Teknologi

- Python
- Streamlit
- JSON
